# Issues and improve idea

### the issues I found in current prompting system

1. It mixed what LLM should do and what python should do. Like requiring boolean from the generated content, which is not suitable for LLM and cost-effective option
2. The clarify question end point and the suggest endpoint share individual prompt, which make no sense to me -- the serve a same scanario
3. Each prompt is unique, lack of modular thinking, like we should use interface to ensure the user portfolio information is always been offered to agent

### the great picture I am thinking

The previous product is more about get some suggestion on how to buy grocery, but I found unless we could answer what we want to cook (at least have rough idea) , we can't confidently say this is what I want to buy and cook later.

So the user story flow becomes:

1. User have an idea and maybe some already bought item in their grocery
2. Recommendation with recipe -- you buy these so you can cook those you love
3. Help you buy them quickly -- you can found these items in these department
4. Present the recipe to help user cook (time to cook)
5. Some foods remain, start from step 1

#### Reference

To achieve above product, I want to use reference like this:

1. Where does the product information come from: use combination as Costco + another community market, use Costco to buy major part of the grocery especially the protein, use community market to buy vegetables
2. What workflow we should use to buy grocery: The workflow frame came from Ethan Chlebowski's YouTube channel, I like his idea and plan to embedded the frame he suggests in my product, which are:
   1. For what type of food we should buy for daily meal preparation: PCSV framework
   2. For how to buy groceries: two-trip grocery system
   3. You could find the transcript of the this exact video and his website in appendix
3. Where does the recipe come from: the backbone comes from [J. Kenji López-Alt](https://frienji.kenjilopezalt.com/) and the rest should be material generated by LLM. Reference to 2 books written by Kenji, since I got many ideas from watching Kenji's cooking video: The Food Lab : Better Home Cooking Through Science; The Wok: Recipes and Techniques : Recipes and Techniques.
